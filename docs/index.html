  • Internal database supports thousands of contacts.

  • Automated voice playback of the date and time of the appointment.

Data Integration 
AutoCall integrates with your existing office software systems. There is no need to re-enter your contacts information.  AutoCall can suck many types of data files directly into the AutoCall database.  With AutoCall you can import and export data to your personally defined file structure.  Even if you have a home gown software solution, the built in Schema Wizard allows you to create a custom link between AutoCall and your data. And, for a nominal fee, our data integration experts are always available to create your custom data schemas for you.

The AutoCall import features has a syncronization option.  This option allows AutoCall to only update data that matches the data being imported.  Data can be syncronized based on an external account number of on the first and last name of the contact.  With syncronization, you never lose additional contact information when you import updated contact data.
